About starCAT

starCAT is an algorithm to annotate single-cell transcriptomes with pre-defined catalogs of reference
gene expression programs (GEPs). Read more about starCAT in our preprint.

This website hosts reference GEP catalogs created by ourselves and others. Learn about and download
available reference catalogs here

For datasets less than 2GB in memory, you can upload your data to run starCAT directly here

For larger datasets, please download our software from github to run starCAT locally.
